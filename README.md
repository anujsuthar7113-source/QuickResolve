# QuickResolve

## Overview

QuickResolve is a full-stack web-based Complaint Management System developed to streamline complaint handling in organizations such as colleges, offices, and institutions.
The system provides role-based access for Users, Administrators, and Departments with automated complaint routing and real-time tracking.

## Features

- User Registration & Login
- Role-Based Authentication
- Complaint Submission
- Smart Complaint Routing
- Complaint Tracking
- Admin Dashboard
- Department Dashboard
- Analytics Dashboard
- Feedback System
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- PHP
- MySQL
- XAMPP
- Chart.js

## Modules

### User
- Register/Login
- Submit Complaint
- Track Complaint
- View History
- Give Feedback

### Admin
- Manage Users
- Manage Departments
- Assign Complaints
- View Analytics

### Department
- View Assigned Complaints
- Update Status
- Resolve Complaints

## Installation

1. Install XAMPP.
2. Copy project into htdocs.
3. Import database.sql into phpMyAdmin.
4. Start Apache & MySQL.
5. Open:

http://localhost/QuickResolve
